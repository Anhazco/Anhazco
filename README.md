Soy un desarrollador **Full Stack** apasionado por la creación de sistemas robustos y eficientes. Me especializo en el desarrollo backend utilizando **C# (.NET)** y **Java**, y en la creación de interfaces web modernas y de alto rendimiento con **Astro** y **TypeScript**.

- 💻 Actualmente enfocado en arquitecturas escalables y desarrollo web moderno.
- ⚡ Me encanta automatizar procesos y diseñar bases de datos optimizadas.
